1. How to register Exchangily account ?
Exchangily testnet is a testing network for testing core system development by core team, or testing application by application developers, coins in testnet is for test purpose only, no any usage and value and unable being used in production network.
Generally, testnet is with all features of production network, however, it may have more and newer features compare to production network.

2. What is testnet seed node ? How to check testnet state?
Exchangily official testnet seeds are: 35.182.160.212 and 13.59.134.49, if you know other nodes' ip, please take them for priority.
To check status of testnet blockchain, please visit link: fabtest.info

3. About Exchangily production network
Exchangily production network is the official Exchangily network, Exchangily coins in Exchangily production network is the official Exchangily coin, it is usable and valuable as whitepaper outlined.
According to Exchangily development plan, Exchangily production network will conduct a hard fork named "The Force" by the end of Sep. 2018, new netwrok after "The Force" day will be with full features designed in whitepaper, that is, will meet enperprise grade needs.

4. How to connect to production network? how to check production network status?
You can connect to Exchangily Foundation official seed nodes directly: 54.215.244.48 and 18.130.6.117, you may connect to any other nodes also if you know.
To check status of Exchangily production network, please visit link: fabexplorer.info

5. What kind wallet available?
Currently we have Ubentu version and windows wallet are available, web wallet or light wallet will be available in 2 weeks.

1. 如何注册亿币账户 ？
亿币测试网络是发链核心开发组用于测试系统功能的网络，大众也可以通过测试网络测试应用软件的功能，测试网络上的币没有任何实用价值，仅供系统及应用功能测试。
一般来说，测试网络具备生产网络的全部功能，尤其对于应用开发来说，在测试网络上稳定运行的仅执行生产网络功能的应用，应可以在生产网络上运行。但测试网络可能具有比生产网络更新及更多的功能，这些功能我们会在开发文档中作出说明。

2. 测试网络有哪些节点可供连接？如何查看测试网络的状态？
在亿币网络域名服务器开通之前，请通过如下IP连接测试网络节点：35.182.160.212 及 13.59.134.49, 如果你知道其它节点，请优先使用其它节点IP。
如果想查看测试网络状态，请通过如下链接查看：fabtest.info

3. 关于亿币生产网络
亿币生产网络是发链系统的正式生产平台，生产网络上的发币是发链的官方货币，具有根据白皮书设定的功能及使用价值。
根据官方预定的开发计划，发链生产网络预计在2018年9月进行一次名为“原力(The Force)”的硬分叉行动，在原力日之后，发链将真正展示原始设计的力量: 满足企业级对去中心化应用的性能要求。

4. 如何连接生产网络？如果查看生产网络状态？
目前官方提供的种子节点为：54.215.244.48 及 18.130.6.117，以后会通过网络域名服务器提供动态联接IP服务。
如果想查看生产网络区块链状态，请访问如下链接: fabexplorer.info

5. 目前亿币有什么版本的主链钱包？
目前已经发布的发链主链钱包有Ubuntu版本及Windows版本，Web版本及轻量钱包将在两周内发布。
